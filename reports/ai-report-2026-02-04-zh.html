Category: 行業 (Industry) 高

概要： 模型上下文協定 (Model Context Protocol, MCP) 目前隸屬於 Linux Foundation，已成為「AI 整合界的 USB-C」。它提供了一種開放標準，透過結構化的 JSON I/O 將 LLM 連接到外部工具和資料庫，有效地消除了關於 API 端點的幻覺。MCP 目前驅動著超過 10,000 個生產伺服器，並已被包括 Zapier 在內的主要參與者採用，後者整合了 8,000 多個應用程式。最近的進展包括 MultiversX 發布了用於區塊鏈錢包管理的原生 MCP 伺服器，以及爆紅的「rentahuman.ai」服務，該服務允許代理透過單次 MCP 調用雇用人類執行現實世界的任務。

背景： MCP 最初由 Anthropic 開發，透過允許任何模型與任何工具對話而無需自定義「膠水」程式碼，解決了「N×M 整合問題」。
