Category: Policy

The 'Clinejection' attack represents a significant supply-chain security vulnerability in AI development tools. This attack poisoned GitHub Actions cache, leaked npm publishing tokens, and resulted in a malicious Cline version being downloaded approximately 4,000 times before takedown. Background: Prompt injection and supply chain attacks represent emerging attack vectors in AI agent systems as these tools become more autonomous and integrated into developer workflows. The Clinejection attack demonstrates how seemingly benign inputs like GitHub issue titles can be weaponized to compromise AI agents, which unlike traditional software, can interpret and act on natural language instructions embedded in user content.
